# Exploratory Data Analysis (EDA) of Work-Life Balance and Longevity Analysis

This exploratory data analysis project analyzes how daily lifestyle choices (work, rest, sleep, and exercise) affect longevity, using Python and Excel to explore patterns across 10,000 individuals.

The core question needed to be answer in this project is: 

> How do daily lifestyle choices, including work, rest, sleep, and exercise, affect how long a person lives?

---

## Dataset Overview

This synthetic dataset contains information on 10000 individuals and explores the relationship between age, occupation, daily time allocation (work, rest, sleep, exercise) and longevity. The data is designed to support regression modeling and analysis of how lifestyle choices impact lifespan.

The dataset captures realistic patterns where daily activities sum to approximately 24 hours, and includes outliers representing extreme work or rest patterns. This makes it ideal for exploring non-linear relationships and diminishing returns in health outcomes.

The dataset is uploaded by [Oluwatishin Adewale on Kaggle](https://www.kaggle.com/datasets/oluwatosinadewale/quality-of-life-data).
